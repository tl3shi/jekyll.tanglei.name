About Me

目前是一名后台开发工程师 @宜信大数据创新中心, 毕业于中南大学和清华大学, 曾工作于腾讯 和 友盟. 我的大致经历如下:

  • 2008年入读于湖南长沙的中南大学软件学院软件工程专业.(很多人不知道中南大学是在长沙滴, 在此强调下)
  • 2010年5月通过软考软件设计师, 同年10月通过软考高级——系统架构设计师, 并取得湖南省第二名.
  • 本科期间喜欢搞java. 做过这些小玩意儿(java桌面开发(仿QQ), java web开发(SNS), java android开发(MSNS)).
  • 2011年7月–2012年1月, 在腾讯(深圳总部)实习, 就职于B2互联网业务系统/社交平台部/社区产品中心/社区开发组/基础功能组/, 捣鼓起了php, 参与了朋友网的相关开发;
  • 2011年10月, 获得清华大学软件学院工学硕士的推免offer. 毕设跟着李春平老师做软件缺陷预测方面的研究, 准备在研究生阶段研究下数据挖掘……
  • 2012年7月–2012年9月, 在友盟Umeng实习, 参与umtrack后台开发, 捣鼓起了python;
  • 2012年8月就读于清华大学软院学院, 阴差阳错在计算机图形学与辅助设计研究所(图所 in short), 跟着雍俊海老师做研究.
  • 2015年7月硕士毕业, 硕士学位论文获清华大学优秀硕士学位论文, 同年入职于宜信大数据创新中心.

其实还有个爱好, 吹竹笛, 虽然技术不咋滴. 成立了清华大学笛子协会ThuFlute.
